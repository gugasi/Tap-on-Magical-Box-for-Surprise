# Tap on Magical Box for Surprise

This is a simple and cute interactive web application where clicking on a magical box reveals a surprise from a cat and a dog.

## How To Use

1.  Clone or download the repository.
2.  Open the `index.html` file in your web browser.
3.  Click on the magical box to see the surprise!

## Features

* **Interactive Magical Box**: A beautifully styled box that invites the user to click on it.
* **Animated Characters**: When the box is clicked, a cat and a dog pop up with animations.
* **Cute Surprises**:
    * The cat, wearing a party hat, offers a bouquet of peonies and a sweet message.
    * The dog, dressed as a knight, presents a rose with a charming compliment.
* **Dynamic Background**: The scene is set against a dynamic night sky with:
    * A glowing moon.
    * Twinkling stars that are randomly generated.
    * Animated falling stars.
* **CSS Animations**: The project uses CSS for various animations, including sparkling effects on the box and blinking eyes for the characters.

## Files

* `index.html`: The main HTML file that structures the web page.
* `style.css`: This file contains all the styles and animations for the elements on the page.
* `sky.js`: This script is responsible for creating the dynamic and animated night sky background.
* `magic-box.png`: The favicon for the web page.
